# Netflix Data Analysis

## Project Overview
Analyzed 8807 Netflix titles to find insights about content trends.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
1. Netflix has 6131 Movies vs 2676 TV Shows
2. USA is the top content producer, followed by India
3. TV-MA is the most common rating — Netflix targets adults
4. Content additions peaked in 2018
